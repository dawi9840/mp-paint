# mp-paint
Using mediapipe hand detection to create a virtual painting.

## Install  

**Conda virtual env**  
```bash

conda create --name [env_name]  python=3.8
conda activate [env_name]
pip install numpy
pip install mediapipe
pip install opencv-python
```
